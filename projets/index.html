Knowledge⌗

knowledge vous permet de construire une base de connaissance à partir de simples fichiers textes.

En utilisant la syntaxe markdown pour la mise en forme, créer des liens entre les documents etc. notre solution reste complètement ouverte : stockez, éditez et partagez vos documents avec les solutions dont vous avez déjà l’habitude : vim + git pour les devs, notepad ou dillinger et google drive pour le plus grand public.

Télécharger la version linux
